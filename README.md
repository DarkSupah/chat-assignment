# chat-assignment

Тестовое задание по чату на основе matrix-js-sdk

## Установка проекта
В проекте используется yarn как менеджер зависимостей

Если он не установлен, установите
```sh
npm -g yarn
````

Для установки зависимостей
```sh
yarn
```

### Запуск проекта

```sh
yarn dev
```

### Сборка

```sh
yarn build
```

### Линтер

```sh
yarn lint
```
